# Project Statement: Restaurant Bill Calculator

## Introduction
This project is a program that displays a menu, takes coustomers orders, and calculates the total bill. It uses basic data structures like lists and tuples

## Objective
To build a small restaurant billing system that:
- Shows a menu
- Lets coustomer pick items by number
- Adds chosen items to a bill
- Prints the total amount at the end

## Technologies Used 
- Python 3
- Tuples (for menu items)
- Lists (for storing orders)

## Method
1. A menu is created using tuples.
2. User enters item numbers to order.
3. Valid choices are added to a list.
4. Program sums the prices and prints the receipt.

## Conclusion
This project demonstrates simple billing logic and Python basics in an easy, beginner-friendly way.
